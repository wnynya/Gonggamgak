  } 
}
웹소켓을 통해 나가는 데이터
가습기 제어 
{ 
  "event": "microwave-humid", 
  "data": { 
    "to": 1 // 전자레인지 번호
    "index": 1 // 가습기 번호 (1 또는 2)
    "on": true // 켜려면 true 끄려면 false 
  } 
}
화면 표시 텍스트 제어 
{ 
  "event": "microwave-text", 
  "data": { 
    "to": 1 // 전자레인지 번호
    "text": 'text' // 표시 문구 (a-z0-9)
  } 
}

  "data": { 
    "to": 1 // 전자레인지 번호
    "on": true // 켜려면 true 끄려면 false 
  } 
}
