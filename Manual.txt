[오픈]
1. 디스플레이 켜기
  1.1. TV 전원 켜기 (아래쪽 TV 앞, 그램 뷰 뒤의 TV 리모컨 사용)
  1.2. 그램 뷰 전원 켜기 (화면 좌측 전원 버튼, 화면 안 나오면 좌측 선 뺏다 꽂기)
  1.3. 프로젝터 전원 켜기 (아래쪽 TV 앞, 그램 뷰 뒤의 프로젝터 리모컨 사용)
  1.4. 전자레인지 안 테블릿 화면 켜기 (전면 가장 오른쪽 버튼, 후 화면 잠금해제)
  1.5. 세탁기 안 디스플레이 켜기 (좌측 벽 콘센트 꽂기)
  * 그램, 미니PC, 틔운은 켜져 있는 상태이므로 따로 전원 조작하지 않아도 됩니다.
2. 화면 표시하기
  기본적으로 화면 전원만 켜면 컴퓨터에 연결된 화면이 바로 표시되어야 합니다.
  화면 표시되지 않는 경우 아래 표시 방법 참조 부탁드립니다.
  2.1. 컴퓨터에 연결된 화면 표시하기
    모든 컴퓨터는 와그작 구글 계정으로 원격 접속 및 조작 가능합니다.
    2.1.1. 그램 14-1 (메인 서버, 전자레인지) 
      * 복잡해요
      2.1.1.1. 서버 켜기
        충돌 방지 위해 모든 CMD 창 꺼 주세요.
        바탕 화면 Gonggamgak -> script 폴더로 이동해주세요.
          start-main-bridge 실행해주세요.
          start-main-app 실행해주세요.
          start-microwave-bridge-1 실행해주세요.
          start-microwave-bridge-2 실행해주세요.
      2.1.1.2. 전자레인지 안 태블릿 화면 표시 준비하기
        크롬 브라우저로 [https://ggg.ccc.vg/tab/unity1/] (탭1) 주소 접속해주세요.
        크롬 브라우저로 [https://ggg.ccc.vg/tab/source1/] (탭2) 주소 접속해주세요. 화면 클릭 후 탭 선택에서 아까 연 탭 (탭1) 선택해주세요.
      2.1.1.3. 틔운 화면 표시하기
        * 2.1.1.2. 와 다른 브라우저 사용해야 화면이 깔끔하게 표시됩니다.
        엣지 브라우저로 [https://jiiii-09.github.io/teun/] 주소 접속 후 전체화면으로 띄워주세요. (오른쪽으로 밀기, 틔운에 표시)
      2.1.2.3. 다 한 후 unity1 (탭1) 에 마우스 클릭해사 포커스 해주세요!
    2.1.2. 그램 16-1 (TV, 터디)
      2.1.2.1. 전자레인지 안 태블릿 화면 표시 준비하기
        크롬 브라우저로 [https://ggg.ccc.vg/tab/unity2/] (탭1) 주소 접속해주세요.
        크롬 브라우저로 [https://ggg.ccc.vg/tab/source2/] (탭2) 주소 접속해주세요. 화면 클릭 후 탭 선택에서 아까 연 탭 (탭1) 선택해주세요.
      2.1.2.2. TV 터치디자이너 띄우기
        바탕화면 -> 와그작_모니터1_송출용.toe 파일 열기 -> m1, m2 노드 클릭(하나씩) -> 오른쪽 창에서 Open as seperate window에 open 버튼 클릭
      2.1.2.3. 다 한 후 unity2 (탭1) 에 마우스 클릭해사 포커스 해주세요!
    2.1.3. 그램 16-2 (시각)
      크롬 브라우저로 [https://ggg.ccc.vg/input/sight/] 주소 접속 후 카메라 시작해주세요. 이 창은 띄운 채 가만히 두시면 됩니다.
      크롬 브라우저로 [https://ggg.ccc.vg/raw/sight/] 주소 접속 후 전체화면으로 띄워주세요.
    2.1.4. 그램 16-3 (청각)
      크롬 브라우저로 [https://ggg.ccc.vg/raw/hearing/] 주소 접속 후 전체화면으로 띄워주세요.
    2.1.5. 그램 16-4 (촉각)
      충돌 방지 위해 모든 CMD 창 꺼 주세요.
      바탕 화면 Gonggamgak -> script 폴더로 이동해주세요.
        start-touch-bridge 실행해주세요.
      크롬 브라우저로 [https://ggg.ccc.vg/raw/touch/] 주소 접속 후 전체화면으로 띄워주세요.
    2.1.6. 미니 PC (프로젝터)
  2.2. 전자레인지 안 태블릿 화면 표시하기
    * 페이지 접속하면 우측 상단에 전체화면 버튼 있습니다.
    (오른쪽) 크롬 브라우저로 [https://ggg.ccc.vg/tab/stream1/] 주소 접속 후 전체화면으로 띄워주세요.
    (왼쪽) 크롬 브라우저로 [https://ggg.ccc.vg/tab/stream1/] 주소 접속 후 전체화면으로 띄워주세요.
  2.3. 세탁기 안 디스플레이 화면 표시하기
    전원 켜지면 자동으로 표시됩니다.
3. 끈 조명 켜기
  끈 조명의 전원 스위치 (파란 LED) 터치하면 켜집니다. 앞쪽 멀티탭 주변에 있습니다.

  1.1. TV 전원 끄기 (아래쪽 TV 앞, 그램 뷰 뒤의 TV 리모컨 사용)
  1.2. 그램 뷰 전원 끄기 (화면 좌측 전원 버튼)
  1.3. 프로젝터 전원 끄기 (아래쪽 TV 앞, 그램 뷰 뒤의 프로젝터 리모컨 사용)
  1.4. 전자레인지 안 테블릿 화면 끄기 (전면 가장 오른쪽 버튼)

  끈 조명의 전원 스위치 터치하면 꺼집니다. 앞쪽 멀티탭 주변에 있습니다.
3. 틔운 물 채우기
  3.1. 틔운 중앙 화면 [물 보충 필요]에 흰색 불이 켜진 경우에 물 채워주세요.
  3.2. 전선 걸리지 않게 틔운 문 열기
  3.3. 식물 아래쪽 트레이 당겨서 연 뒤 물통 꺼내기
  3.4. 물통에 물 채운 뒤 다시 넣고 트레이 닫기
  * 틔움 사용법 영상 참고해주세요 [https://youtu.be/tf_4-fss1jM?t=353]
4. 주의사항
  4.1. 멀티탭 전원 끄거나 뽑으시면 안됩니다.
  4.2. 그램, 미니 PC 전원 끄시면 안됩니다. 켜 둔 채 마감해주세요. (데이터 손상 우려 있음)
  4.3. 틔운 전원 끄지 말아주세요.
